\subsection{Detailed Description}
A \hyperlink{classTrajectory}{Trajectory} object contains a vector of \hyperlink{classFrame}{Frame} objects, plus other info on the simulation (number of atoms). It also contains the special xd pointer that libxdrfile needs to open the xtc file, as well as the number of atoms in the system, the number of frames read in, and an \hyperlink{classIndex}{Index} object.
